an(s) vers le passé et vers l'avenir


Guide d'utilisation

Soumettez un texte au format DOCX ou ODT. Le programme recherchera des références à des articles de code. Il vous alertera :

  • Si le texte est introuvable sur Légifrance(abrogé, faute de frappe...)
  • Si le texte a été récemment modifié (période définie par vous)
  • Si le texte va être modifié prochainement (période définie par vous)

Codes actuellement testés :

  • Code civil
  • Code de commerce
  • Code du travail
  • Code de la propriété intellectuelle
  • Code pénal
  • Code de procédure pénale
