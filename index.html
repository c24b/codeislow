Guide d'utilisation

Soumettez un texte au format DOCX ou ODT. Le programme recherchera des références à des articles de code (ex : art. 1240 C. civ.). Il vous alertera :

  • Si le texte est introuvable sur Légifrance (abrogé, faute de frappe...)
  • Si le texte a été récemment modifié (période définie par vous)
  • Si le texte va être modifié prochainement (période définie par vous)

Codes actuellement testés :

  • Code des assurances
  • Code civil
  • Code de commerce
  • Code de la consommation
  • Code pénal
  • Code de procédure civile
  • Code de procédure pénale
  • Code de la propriété intellectuelle
  • Code de la santé publique
  • Code de la sécurité intérieure
  • Code de la sécurité sociale
  • Code du travail
